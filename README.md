# Public-Architecture-for-NHR-AI-Assistant-
This repository contains some code to create a public architecture for NHR AI Assistance

google_colab_code1_new8_gpt2.txt
This code is the main code to create the backend API using Google Colab, through ngrok which would create URL

google_colab_code_001_connect.txt
This code is used to connect (mount) Google Colab to a database or an embedded vector database (ChromaDB)

google_colab_code_002_install.txt
This code is used to install necessary modules, such as chromadb or ngrok

google_script_code5_gpt.txt
This code is Code.gs in the Google Apps Script, which create a bridge between the backend API (Google Colab) and User Interface (UI) or website 

google_script_index2.txt
This code is Index.html in the Google Apps Script, which created the UI in the website
